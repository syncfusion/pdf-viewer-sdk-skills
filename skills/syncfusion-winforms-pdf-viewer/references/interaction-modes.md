# Interaction Modes

PDF Viewer supports two cursor modes for interacting with PDF documents via the `CursorMode` property of `PdfViewerControl`.

## Selection Mode

```csharp
pdfViewerControl1.CursorMode = PdfViewerCursorMode.SelectTool;
```

## Panning Mode

```csharp
pdfViewerControl1.CursorMode = PdfViewerCursorMode.HandTool;
```
